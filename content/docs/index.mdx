Next Auth Roles Template is **Open Source**.

Built on [Next SaaS Stripe Starter](https://github.com/mickasmt/next-saas-stripe-starter), it integrates Next.js 14, Prisma, Neon, Auth.js v5, Resend, React Email, Shadcn/ui.

With **Authentification, User roles & Dashboard**, you get a solid foundation for your future project.

This documentation is your go-to resource for configuring and using the template effectively.

Let's get started and happy coding!
